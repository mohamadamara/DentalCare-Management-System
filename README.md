# DentalCare Management System

This is a Java desktop application for managing dental clinic operations.

## Features

- Patient management
- Appointment scheduling
- Inventory management
- Database integration
- GUI interface

## Technologies used

- Java
- Swing
- JDBC
- Microsoft Access

## Author

## Reporting (JasperReports)
- Patient Full Report: generates a complete report for a selected patient including personal info, medical history, insurance, and treatments.
- Treatments by Doctor Report: generates a report filtered by doctor/dentist showing treatments performed and related details.
- Reports are generated from the database and displayed on screen.
